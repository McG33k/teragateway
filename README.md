# Teragateway WordPress Theme

A custom WordPress theme built for **Teragateway**, an IT Consulting Services firm.  
This theme is designed to showcase consulting services with a professional, modern layout and seamless plugin integration.

## ✨ Features
- Responsive design for desktop, tablet, and mobile
- Customizable homepage with service sections
- Integrated "Learn More" and call-to-action buttons
- Support for popular WordPress plugins
- Easy setup and configuration

## 📂 Installation
1. Download or clone this repository:
   ```bash
   git clone https://github.com/mcg33k/teragateway.git

2. Copy the theme folder into your WordPress installation under:
   wp-content/themes/
   
3.Activate the theme from the WordPress Admin Dashboard:
- Go to Appearance → Themes
- Select Teragateway Theme and click Activate

4. 🛠️ Requirements
- WordPress 6.0+
- PHP 7.4+
- MySQL 5.7+ or MariaDB equivalent

🚀 Getting Started
- Customize the homepage via Appearance → Customize
- Add your services, team, and contact information
- Install recommended plugins for extended functionality (WPForms, Popup Maker)
